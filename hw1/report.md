## Особенности тестовых функций
### Для каждой из функций проведены две серии испытании. В первом случае - значения точные, во втором - зашумлены независимым гауссовским шумом с постоянной дисперсией.
- **Некоторые функции умножаются и сдвигаются на константу, это нужно для улучшения визуализации**

1. **Линейные функции** - базовый тест корректности
2. **Гладкие периодические** - проверка сохранения свойств
3. **Быстрорастущие** - тест на устойчивость
4. **|x|^0.5** - точки излома
5. **Функция Рунге** - склонность к осцилляциям
6. **sin(1/x)** - предельный случай

## Ключевые наблюдения

- **Вариационная схема** показывает себя лучше на зашумлённых данных + сохраняет ывойства исходной фунуции.
- **Трёхточечная схема** Дает более точные значения, на точных данных. (O(h³) против O(h²))
- Для функций с разрывами производных обе схемы дают сравнимую точность
- Осциллирующие функции сложны для обеих схем
- Краевые эффекты сильнее влияют на трёхточечную схему


Графики демонстрируют хорошее соответствие аппроксимаций исходным функциям, за исключением особых случаев (осцилляции, быстрый рост).
