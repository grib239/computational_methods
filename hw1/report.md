## Особенности тестовых функций
### Для каждой из функций проведены две серии испытании. В первом случае - значения точные, во втором - зашумлены независимым гауссовским шумом с постоянной дисперсией.
- **Некоторые функции умножаются и сдвигаются на константу, это нужно для улучшения визуализации**

1. **Линейные функции** - базовый тест корректности
2. **Полином степени 3** - проверка корректности усредняющей схемы (любой локальный метод воспроизводящий сплайн/полином степени $\leq$ b должен глобально воспроизводить полином/сплайн степени $\leq$ b).
4. **x^4** - не должен воспроизводиться.
2. **Гладкие периодические** - проверка сохранения свойств
3. **Быстрорастущие** - тест на устойчивость
4. **|x|^0.5** - точки излома
5. **Функция Рунге** - склонность к осцилляциям
6. **sin(1/x)** - предельный случай

## Что реализовано

Две схемы построения фппроксимации функции кубическими B-сплайнами и среда для тестирования с возможностью менять функции, выбирать разный шаг равномерной сетки, добавлять гауссовский шум.

### Усредняющая схема
Реализована классическая схема локальной аппроксимации на основе кубических B-сплайнов, воспроизводящая полиномы до 3 степени (https://www.uio.no/studier/emner/matnat/math/MAT4170/v22/pensumliste/splinebook-2018.pdf) (стр. 164). коэффициенты `w =  [1.0, -8.0, 20.0, -8.0, 1.0] / 6.0` подобраны так чтобы локально интерполировать 5 точек (по 3 на каждый из центральных отрезков). могут быть найдены из системы:  ![alt text](image.png)

### Точечная схема (variation diminishing)
`coeff = func(center)`
Называется так из за теоретического свойства сплайнов сохранять число вариаций ломанной.

## Ожидаемые результаты

Усредняющая схема воспроизводит полиномы до 3 степени => должна иметь точность $O(x^4)$ для функций из $C^4[a, b]$.
Точечная схема не имеет теоретических оценок на сходимость (лучше чем $O(x^2)$), однако должна лучше показывать себя на шумных данных (исходя из здравого смысла). 

## Ключевые наблюдения

- Результаты экспериментов великолепно соответствуют теории. 

Анализируя гладкие функции (exp, sin) на данных без шума - видно, что при уменьшении шага в $\sqrt{10}$ раз, ошибка точечной схемы уменьшается приблизительно на 1 порядок, ошибка усредняющей схемы на два порядка.
- На более сложных функциях (не гладких), как и на зашумленных данных, лучше себя показывает вариационная схема.
