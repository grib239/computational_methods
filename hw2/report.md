- начинаем с $y_0(x) = f(x)$

- для каждой точки xi вычисляем интеграл $K(x_i, t, y_h(t)) dt$.
- получаем правую часть $RHS_i = f(x_i) + I_i$.
- Решаем линейную систему $A * c_{new} = RHS$.
